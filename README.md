#

Me chamo Pedro Menna, tenho 21 anos e atuo como DevOps Engineer com foco em Infraestrutura, Kubernetes, CI/CD e Cloud. Atualmente trabalho com ambientes corporativos voltados para automação, deploys, containers e sustentação de aplicações críticas.
